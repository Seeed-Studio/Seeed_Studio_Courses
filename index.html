Edge AI 101 with Nvidia Jetson

中文简体

Course Overview

This course is designed for beginners in edge computing, aiming to help students master edge AI technology by introducing hardware platforms, Python programming, and AI project development.

  • Course Platform: reComputer J1020 v2 (Jetson Nano 4GB)
  • Programming Language: Python
  • Tools Used: OpenCV, PyTorch, YOLOv8
  • Project Goal: Intelligent Surveillance System

Course Structure

  1. Basics of Edge AI Programming
  2. Introduction to Computer Vision Applications
  3. Advanced Deep Learning
  4. Intelligent Surveillance System Development

Course Setup and Requirements

Hardware Requirements

  • reComputer J1020 v2 (Jetson Nano 4GB)
  • 128GB NVMe M.2 SSD
  • Logitech C270 HD Camera

Software Requirements

  • JetPack OS (for Jetson platforms)
  • Python 3.x development environment
  • AI libraries: OpenCV, PyTorch, YOLOv8
  • Visual Studio Code (recommended)
